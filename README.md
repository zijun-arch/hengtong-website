# 恒通官网 - 企业官网建设项目

## 项目信息
- **客户**: 恒通冷链公司
- **状态**: ✅ **第一版已完成**（2026-05-21）
- **创建日期**: 2026-05-21

## 设计定位
- 高端大气企业官网
- 网页版 + 手机版（响应式设计）
- 参考顶级网站设计（Stripe、IBM、Apple等）

## 设计要求
- 极简、大留白、大字体
- 首页3秒内说清业务
- 克制动效（滚动动画、微交互）
- 强视觉层次
- 响应式适配

## 网站架构（已完成）

| 板块 | 内容 | 状态 |
|------|------|------|
| **Hero首屏** | 视频背景+标题+4项统计数据 | ✅ |
| **关于我们** | 图文布局+400字公司介绍 | ✅ |
| **核心价值观** | 4卡片：品质/科技/客户/绿色 | ✅ |

| **新闻资讯** | 3篇示范文章 | ✅ |
| **联系CTA** | 渐变背景+电话/邮箱/地址 | ✅ |
| **页脚** | 4列布局+版权信息 | ✅ |

## 技术选型
- 纯HTML+CSS+JS单文件，零外部依赖
- 响应式设计（桌面4列→平板2列→手机1列）
- 图标全部使用emoji
- 所有图片使用CSS渐变/内联SVG
- IntersectionObserver滚动动画
- 配色：深蓝 #0a1628 + 亮蓝 #3a7bd5
